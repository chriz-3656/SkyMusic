YOUR ERROR:
──────────
❌ Invalid Form Body
❌ In components.0.components.0.emoji.id: Invalid emoji
❌ In components.0.components.1.emoji.id: Invalid emoji

QUICK EXPLANATION:
──────────────────
Discord is saying:
  • Component 0, button 0 (PREV emoji) has invalid ID
  • Component 0, button 1 (PAUSE emoji) has invalid ID
  • These emojis don't exist in your Discord app
  • OR their IDs don't match what's in bot/utils/emojis.py

ROOT CAUSE:
──────────
The emoji IDs in bot/utils/emojis.py don't match your Discord app's 
actual emoji IDs.

This happens when:
  ✗ Emojis were re-uploaded (new IDs assigned)
  ✗ Emojis were deleted and added again
  ✗ Wrong IDs were initially pasted
  ✗ Different Discord application was used

SOLUTION OVERVIEW:
──────────────────

Your emojis ARE uploaded to Discord (we can see from screenshots).
You just need to get their CURRENT IDs and update bot/utils/emojis.py

Three ways to fix:

  1️⃣ AUTOMATIC (Easiest)
     → Run: python3 test_emoji_ids.py
     → Automatically fetches all emoji IDs
     → Generates correct Python code
     → No manual copying needed

  2️⃣ MANUAL (Most Reliable)
     → Go to Discord Developer Portal
     → Emojis section
     → Read IDs manually from the table
     → Update bot/utils/emojis.py

  3️⃣ BROWSER CONSOLE (Advanced)
     → Use JavaScript in browser DevTools
     → Extract all emoji names + IDs at once
     → Paste into extract_emoji_ids.py

STEP-BY-STEP AUTOMATIC FIX:
───────────────────────────

1. Get your bot token:
   → Discord Developer Portal
   → Select your app
   → "Bot" section
   → Copy "TOKEN"

2. Set environment variable:
   export DISCORD_TOKEN="paste_token_here"

3. Run auto-extractor:
   python3 test_emoji_ids.py

4. You'll see output like:
   ✅ Found 39 emojis in your Discord app:
   
     play: 1494517456844782842
     pause: 1494517412678708292
     stop: 1494517441357942895
     ... etc

5. Open bot/utils/emojis.py

6. Replace the emoji definitions with the output

7. Save file

8. Restart bot:
   python3 -m bot.main

9. Should see:
   ✅ Successfully synced 10 app commands with Discord

10. Test /play command in Discord - should work!

STEP-BY-STEP MANUAL FIX:
────────────────────────

1. Open Discord Developer Portal:
   https://discord.com/developers/applications

2. Click your "SkyMusic" application

3. Click "Emojis" in left menu

4. You'll see table: EMOJI | UPLOADED BY | EMOJI ID

5. For EACH emoji on Page 1:
   → Read emoji name
   → Read emoji ID (rightmost column)
   → Note it down

6. Open bot/utils/emojis.py in text editor

7. Find each emoji definition and update ID:
   
   Before: PLAY = E("play", 1494517456844782842)
   After:  PLAY = E("play", 1649451750306048331)
   
   (Use the ID from step 5)

8. Do this for ALL emojis on Page 1

9. Save file

10. Restart bot: python3 -m bot.main

11. Should work!

IMPORTANT CHECKS:
─────────────────

When copying emoji IDs, make sure:

  ✓ ID is 18-19 digits
  ✓ No quotes: E("play", 12345) ← WRONG (too short)
  ✓ No commas: E("play", 1,494,517...) ← WRONG
  ✓ No spaces
  ✓ Numbers only
  ✓ Name matches Discord exactly (lowercase)

QUICK FORMAT CHECK:
───────────────────

✅ CORRECT:
   PLAY = E("play", 1494517456844782842)
   PAUSE = E("pause", 1494517412678708292)

❌ WRONG:
   PLAY = E("play", "1494517456844782842")  ← Has quotes
   PLAY = E("PLAY", 1494517456844782842)    ← Uppercase
   PLAY = E("play", 1,494,517,456,844,782,842) ← Has commas
   PLAY = E("play", 12345)                  ← Too short

VERIFICATION:
──────────────

After updating, check if each emoji exists in Discord:

  → Go to Discord Emojis page
  → Search for "play"
  → Should find the emoji
  → ID should match what you put in code

If emoji doesn't exist:
  → It was deleted or never uploaded
  → Need to upload it to Discord first

HOW TO UPLOAD MISSING EMOJI:
─────────────────────────────

1. Go to Discord Developer Portal
2. Your App → Emojis
3. Click "Upload Emoji" button
4. Select image file
5. Type emoji name (must be lowercase)
6. Click Upload
7. Copy the new emoji ID
8. Update bot/utils/emojis.py

AFTER FIXING:

4. Open Discord

5. Try /play command

6. Should see control buttons appear with emojis

7. Click buttons to test
